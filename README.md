# create_ssl
Scripts for create and revoke ssl certificates
В данном репозитории находится два скрипта для автоматизации создания ssl сертификатов и их отзывов. Скрипты создают/отзывают сертификаты как пользователей так и клиентов. Скрипты интерактивные и в процессе надо отвечать на вопросы. В зависимости от ответов вы получите то, что получите. 
ВАЖНО: данные скрипты написаны для двухуровневой иерархии ЦА. 
ВАЖНО: запускать данные скрипты нужно под пользователем, в чьей домашней директории находится ЦА.
